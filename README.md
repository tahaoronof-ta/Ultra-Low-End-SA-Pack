## Overview

**Ultra Low-End SA Pack** is a performance-focused modpack designed specifically for low-end and older PCs. The goal is to maximize FPS, reduce stuttering, and improve stability while preserving the classic GTA San Andreas experience.

Whether you're playing on an old laptop, a budget PC, or simply want smoother gameplay, this modpack helps you get the best possible performance.

## System Requirements

### Minimum

* Windows 7 / 8 / 10 / 11
* 2 GB RAM
* Dual-Core Processor

### Recommended

* 4 GB RAM or higher
* SSD Storage

## Installation

1. Download the latest release from the Releases section.
2. Extract the archive using WinRAR or 7-Zip.
3. Copy all files into your GTA San Andreas installation folder.
4. Launch the game.
5. Enjoy improved performance.
